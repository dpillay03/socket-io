# Repo: Realtime
## Developer: Daniel Pillay

## Description
An Messaging application that allows users to send messages in a chat room

## Install
1. Clone repo
1.  run `npm install`
1.  run `node index.js` or `nodemon` if you have it installed.
1.  Navigate to http://localhost:3000
1. Click Dashboard
1. Type messages


## Technologies
1. React.js
1. Node.js
1. Parcel
1. Socket-io
